Fazendo requisições HTTP no Angular, crie uma aplicação que faça pesquisa de usuários do Github utilizando a api oficial do githib.


1. Faça um campo para pesquisar o usuário pelo seu username e um botão para fazer a pesquisa. 
1.2 Ao clicar, ele mostrará um Dialog do Material com os dados do usuário, como 
1.2.1 foto de perfil, ✅
1.2.2 descrição, ✅
1.2.3 quantidade de seguidores, ✅
1.2.4 quantas pessoas o usuário está seguindo e qualquer outro dado que você desejar. ✅

2. Também recupere os dados dos repositórios e os liste também no Dialog, logo abaixo dos dados do usuário. 
2.1 De cada repositório, 
2.1.2 mostre o nome dele, ✅
2.1.3 a linguagem utilizada, ✅
2.1.4 quantidade de estrelas e, ✅
2.1.5 ao clicar nas infos de um repo, você será redirecionado à página do repositório que você clicou.✅
